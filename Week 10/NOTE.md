# 浏览器工作原理(排版/渲染)

1. 根据浏览器属性进行排版

- Axis
  - Main Axis 主轴
  - Cross Axis 交叉轴

2. 收集元素进 line

3. 计算主轴方向

- 找出所有 Flex 元素
- 把主轴方向的剩余尺寸按照比例分配给这些元素
- 若剩余空间为负数,所有 flex 元素为 0,等比压缩剩余元素

  4.计算交叉轴

5. 绘制单个元素

- 绘制需要依赖一个图形环境
- 使用 npm 包 `images`

6. 绘制 DOM 树

- 递归调用子元素的绘制方法完成 DOM 树的绘制
- 忽略一些节点
- 实际浏览器中,文字绘制是难点,需要依赖字体库
